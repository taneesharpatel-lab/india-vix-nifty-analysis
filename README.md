# India VIX vs Nifty 50 Returns Analysis
Statistical analysis of the relationship between India VIX (fear gauge) 
and Nifty 50 returns across 1,825 trading days (2019–2026).

## Methods
- Pearson Correlation
- OLS Regression
- Percentile Bucket Analysis

## Key Finding
Peak fear periods (VIX top 20th percentile) produced the strongest 
average next-day returns (+0.174%), consistent with the contrarian thesis.

## Tools
Python, yfinance, scipy, statsmodels, seaborn, pandas
